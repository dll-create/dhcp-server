Why DHCP Server?

Built by engineers, for engineers. Every feature solves a real pain point.

🛡️

Network Safety Probe

Automatically detects existing DHCP servers before starting. Prevents accidental IP conflicts in production networks.

⚡

One-Click Start

Select interface, click Start. No config files, no command line. Sensible defaults get you running in seconds.

🌐

Bilingual UI

Full English & 中文 support with one-click toggle. Auto-detects your system language.

📡

Full DHCP Stack

Custom RFC 2131 implementation. DISCOVER → OFFER → REQUEST → ACK. No third-party DHCP libraries.

📊

Real-Time Dashboard

Live lease table, IP pool stats, and streaming server logs via Server-Sent Events.

💻

Desktop App

Optional Electron wrapper for a native macOS experience. Or just use the browser — your choice.
